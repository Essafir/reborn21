# Reborn21 - Landing Page Conférence Live

**[Voir le site en direct](https://reborn21-sigma.vercel.app/)**

## Description
Intégration front-end d'une page de capture (landing page) pour la conférence en ligne Reborn21. Ce projet consiste en la réalisation d'une interface utilisateur "pixel-perfect" basée sur une maquette professionnelle, pensée pour optimiser l'expérience utilisateur et les conversions.

## Technologies Utilisées
- **HTML5** : Structure sémantique et claire.
- **CSS3** : Variables natives, Flexbox, CSS Grid, et Scroll Snap.
- **JavaScript (Vanilla)** : Logique interactive, gestion du DOM et utilisation de l'API Intersection Observer.
- **GSAP & ScrollTrigger** : Gestion des animations complexes et des effets d'apparition au défilement.
- **Déploiement** : Hébergé sur Vercel avec intégration continue via GitHub.

## Fonctionnalités Clés
- **Design 100% Responsive** : Adaptation fluide sur les supports mobiles, tablettes et ordinateurs de bureau.
- **Carrousel Interactif (Coverflow)** : Implémentation personnalisée en JavaScript pur pour un slider de témoignages en boucle infinie.
- **Animations au Scroll** : Entrées en cascade et translations dynamiques synchronisées avec le défilement de l'utilisateur.
